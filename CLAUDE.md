# Project Context

## Purpose
This tool generates timesheets from JIRA activity for WBSO compliance reporting.

## WBSO (Dutch R&D Subsidy Program)
- **WBSO** = Wet Bevordering Speur- en Ontwikkelingswerk (Dutch R&D tax credit)
- Requires thorough, auditable time tracking documentation
- ~50% of weekly hours should be allocated to WBSO-eligible work
- WBSO entries need detailed descriptions tied to actual JIRA tickets

## Timesheet Format
- Weeks run Saturday to Friday (S, Su, M, T, W, Th, F)
- 40 hours/week, 8 hours/day (Mon-Fri only)
- Weekend columns (S, Su) are always 0
